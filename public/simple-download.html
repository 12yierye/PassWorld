人脸识别模型下载助手

说明：点击下面的链接下载模型文件，并将它们保存到 e:\PassWorld\public\models\ 目录中。

重要：确保文件夹 public/models 存在，如果不存在请手动创建。

1. Tiny Face Detector 模型

用于快速检测图像中的人脸位置

manifest.json shard1 shard2

2. Face Landmark 68 模型

用于识别人脸上的68个关键点

manifest.json shard1

3. Face Recognition 模型

用于识别人脸并提取特征向量

manifest.json shard1 shard2 shard3

4. Face Expression 模型

用于识别人脸表情

manifest.json shard1

完成步骤：

  1. 点击上述所有链接下载模型文件
  2. 将下载的文件保存到 e:\PassWorld\public\models\ 目录
  3. 启动应用: npm run dev
  4. 应用将自动加载模型并提供人脸识别功能

如果无法下载：

  1. 尝试运行命令 node download_models.js 自动下载
  2. 访问 GitHub 原始地址下载：https://github.com/justadudewhohacks/face-api.js/tree/master/weights
  3. 使用代理或VPN访问国外资源
